This website is build based on the work carried out by Robin Chew in the Python WA workshop;2020.


To use this website, you need to make selection of fuel type and a suburb you are looking for and then submit the button.


Then, the backend Flask library(which is one of libraries of Python) will create a URL, in which will be used to pull the data. In the next step, extracted data will be sorted and tabulated and published in the website.
